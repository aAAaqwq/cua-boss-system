# BOSS直聘智能招聘助手 · 使用手册

> 写给老板看的「说人话」版手册。你**不用懂技术、不用敲代码**，
> 只要像跟助理聊天一样，把要做的事告诉智能体，它会自己去 BOSS直聘上点鼠标、发消息、整理简历。

---

## 一、这套系统能帮你做什么

把它想成一个**不知疲倦的招聘助理**。你睡觉的时候，它也能帮你：

| 它能做的事 | 相当于一个真人 HR 在做 |
|---|---|
| 🤝 **主动打招呼** | 在「推荐牛人」里一个个挑符合条件的人，逐个打招呼 |
| 📥 **收集简历和微信** | 把聊过的候选人，简历下载存档、**发起加微信请求**（对方同意才算加上），整理成一张表 |
| 💬 **智能回复消息** | 替你回复候选人的问题，会看聊天记录、不重复问已经知道的事 |
| ⭐ **给候选人打分排名** | 像资深 HR 一样审简历，结合岗位要求打分，告诉你「谁最值得面」 |
| 📅 **预约面试** | 帮你给候选人发面试邀请，选好线上/线下、日期、时间 |
| ⏰ **面试提醒** | 每天提醒你「今天/明天有哪些面试」 |

**核心优势**：筛选标准（哪些学校、什么学历）、回复话术、打分规则，全都可以按你的要求调整，越用越懂你。

## 二、开始前的准备（只需做一次）

这些大多由你的技术同事帮你提前弄好，你只要确认这几件事：

1. ✅ **电脑是 Mac**（苹果电脑），这套系统只能在 Mac 上跑。
2. ✅ **Chrome 浏览器已经登录了你的 BOSS直聘老板账号**。
   - 智能体是「借用」你登录好的浏览器去操作，所以必须先登录。
3. ✅ **干活的时候，别让电脑息屏/睡眠**。
   - 它在帮你点鼠标，屏幕黑了就停了。建议接上电源。
4. ✅ **它干活时，你先别用鼠标去抢 Chrome**。
   - 让它专心操作，你可以去忙别的，回来看结果就行。

> 如果哪天提示「找不到窗口」或「未登录」，多半是 Chrome 退出登录了，
> 重新打开 Chrome 登录一下 BOSS 就好。

## 三、怎么用？——你只管「说」，它来「做」

你和智能体之间就是**聊天**。你用大白话说要干啥，它就去干。

**举个例子**：

> **你说**：「帮我把今天的招聘跑一遍，主动打招呼 20 个人，收集 5 份简历，再智能回复 5 个人。」
>
> **它会做**：自动打开 BOSS → 在推荐页打招呼 → 进沟通页收简历加微信 → 逐个回复消息，
> 全程给你播报进度，最后汇报**每一步实际完成了多少**（真正打招呼几人、收到几份简历、回复了几人）。

你不需要记任何命令，**下面这些话直接照着说就行**👇

## 四、最常用的几句话（照着说即可）

### 🚀 1. 跑一整套流程（最常用）

> 「**帮我跑一遍完整招聘流程**」
> 「打招呼 20 个、收集 5 份简历、回复 5 个人」
> 「全自动跑一遍，学历要求硕士以上」

它会按 **打招呼 → 收集 → 回复** 的顺序一步步做完。

> 💡 **关于数字**（两类含义，别混淆；默认都是 20）：
> - **打招呼 20 = 实实在在打满 20 个人**。它会自动多翻一些卡片，把不符合你筛选标准（学校/学历）的跳过，**直到成功打招呼满 20 人**才停（候选人不够时会提前停并告诉你）。
> - **收集 20 / 回复 20 = 「从聊天列表最上面往下数 20 个联系人」逐个处理**（不是「收到 20 份简历」「回复 20 个人」）。这 20 个里：被学校/学历筛掉的会点「不合适」、没传简历的只记基本信息、已回过的会跳过——**都算在这 20 个之内**。所以**真正收到的简历数 / 回复出去的人数，通常少于 20**，这是正常的。

### ⭐ 2. 看谁最值得面试（看排行榜）

> 「**给我看最近两天的候选人评分排行榜，前 10 名**」
> 「谁最合适？打个分排个名」
> 「最近三天最好的候选人是谁」

它会给你一张排行榜：**谁分高、什么评级、一句话点评**。
（已经面过的人会自动从榜单里去掉，省得重复。）

> 💡 其实**收集简历的时候，AI 就已经自动给每个有简历的人打过分了**，所以看排行榜一般**立刻就有结果**。
> 万一有人还没分（比如刚收集完没来得及），看排行榜时会自动补上。

### 📅 3. 约面试

> 「**给排名第一的候选人约一个线上面试，明天下午两点半**」
> 「帮 XXX 约线下面试，6 月 25 号上午 10 点」

它会去发面试邀请并填好时间。约成功后这个人会自动标记为「已面试」。

### ⏰ 4. 看今天/明天有哪些面试

> 「**今天有哪些面试？**」
> 「提醒我未来三天的面试安排」

纯查看，不会动 BOSS，随时可问。

### 🤝 5. 只想单独做某一件事

| 你想干啥 | 你就说 |
|---|---|
| 只打招呼 | 「帮我在推荐页打招呼 30 个人」 |
| 只收简历 | 「帮我收集 10 份简历和微信」 |
| 只回消息 | 「帮我把未读消息都智能回复一下」 |
| 更新岗位信息 | 「同步一下我 BOSS 上的最新岗位」 |
| 查/导出候选人 | 「把候选人名单导出成表格」（导出全部候选人；想只看有简历的，可让它先筛选再看） |

### 👀 6. 不确定？先「预览」一下

> 「**先预览，别真的操作**」 或 「先 dry-run 跑一遍给我看」

预览会把「准备怎么做、要回复什么」演示给你看。不同步骤的预览**安全程度不一样**，要分清：

- 🟢 **打招呼**、**智能回复**：纯预览，**完全不会发出去**（回复内容只打印给你看，不发送）。最适合先预览。
- 🟡 **收集**：预览**不会写入名单、也不会点「不合适」**，但**仍会照常打开简历、并尝试发起加微信请求**——所以收集这步的预览**不是「完全不动」**，介意的话就别对收集单独预览。

第一次用、或调整了话术/筛选标准后，建议先对**打招呼/回复**这样预览确认。

## 五、推荐的一天用法（节奏参考）

| 时间 | 你说的话 | 目的 |
|---|---|---|
| 上午 | 「跑一遍完整流程，打招呼 30、收集 10、回复 10」 | 广撒网 + 收简历 + 回消息 |
| 中午 | 「看最近两天评分排行榜前 10」 | 挑出值得面的人 |
| 下午 | 「给前三名各约一个线上面试」 | 安排面试 |
| 第二天早 | 「今天有哪些面试？」 | 不漏面 |

> 也可以让技术同事把「每天上午自动跑流程 + 每天早 6 点面试提醒」设成**定时任务**，
> 这样连说都不用说，它自己按点干活。

## 六、怎么看「评分排行榜」

排行榜大概长这样：

```
  #  姓名        学校          学历    总分    评级
  1  张三        清华大学      硕士    88.5   S 强烈推荐
       └ 项目经历扎实，主导过高并发系统，与岗位高度匹配
  2  李四        浙江大学      本科    73.0   A 推荐
       └ 经验对口但缺乏量化成果，可面试进一步确认
```

**怎么读**：

- **总分**：满分 100，越高越好。
- **评级**：一眼判断要不要面 ——

  | 评级 | 含义 | 建议 |
  |---|---|---|
  | **S** | 强烈推荐 | 优先约面 |
  | **A** | 推荐 | 值得约面 |
  | **B** | 可考虑 | 看情况 |
  | **C** | 待定 | 再观察 |
  | **D** | 不推荐 | 可放弃 |

- **下面那行小字**：AI 给的一句话点评，说明亮点和风险。

> **打分依据什么？** 主要看**简历附件 + 岗位要求**。AI 会结合你这个岗位需要什么，
> 重点分析候选人的**项目经历**（做过什么、本人贡献多大、有没有实打实的成果），
> 而不是只看他堆了多少技能词。**没有简历附件的人不会被打分**（缺主要依据）。

## 七、安全与注意事项

- 🐢 **它故意放慢手脚**：每个动作之间会随机停 1～4 秒，模仿真人，避免被 BOSS 判定为机器人。所以它不会「秒做完」，这是正常的。
- 🛑 **遇到当天上限会自动停**：BOSS 对打招呼有每日上限，碰到了它会自己停下并告诉你。
- 🙅 **干活时别手动抢 Chrome**：你同时操作鼠标会打乱它。
- 🔒 **数据存在哪儿**：简历文件、电话、微信、聊天记录都保存在**你自己的电脑**里（不会公开、不会发到招聘平台以外的第三方网站）。
  - **唯一例外**：用到 AI 的两个功能——**智能回复**和**打分排名**——会把候选人的**简历文字和聊天内容**发送给 AI 服务商（DeepSeek）做分析，这是使用 AI 的必要环节（和你平时用 AI 工具一样）。简历的 **PDF 原文件**和**数据库**始终只在本机，不外传。
- 👀 **拿不准就先预览**：调整过设置后，先说「先预览」确认无误，再正式跑。

## 八、遇到问题怎么办（常见情况）

| 现象 | 多半是因为 | 怎么办 |
|---|---|---|
| 提示「找不到窗口 / 未登录」 | Chrome 没开或掉登录了 | 打开 Chrome，重新登录 BOSS 直聘 |
| 回复质量一般、像模板 | AI 回复服务（密钥）没配好 | 找技术同事确认 DeepSeek 密钥已配置 |
| 排行榜里某人没分 | 他还没有简历附件 | 先收集到简历，再看排行榜 |
| 跑到一半停了 | 碰到当日上限 / 网络波动 | 它会说明原因；歇一会儿再让它「接着跑没做完的」 |
| 想改「只要 985/211」「只要硕士」 | 筛选标准要调整 | 跟智能体说「我只要 985 硕士以上」，或让技术同事改筛选配置 |
| 想改回复话术的语气 | 话术/人设要调整 | 跟技术同事说，改一个文字文件即可，**不用改代码** |

> 任何看不懂的报错，**直接把它截图发给智能体问「这是什么意思、怎么办」**，它会帮你解释。

## 九、小词典（看到这些词不用慌）

| 词 | 大白话 |
|---|---|
| **智能体 / Agent** | 就是帮你干活的 AI 助理，你跟它聊天下指令 |
| **打招呼** | 在推荐页主动联系候选人，相当于「投出第一句话」 |
| **收集 / 收简历** | 把简历下载存档、发起加微信请求（对方同意才加上）、整理成表 |
| **智能沟通 / 回复** | 替你回复候选人消息，会看上下文、不啰嗦 |
| **评分 / 排行榜** | AI 给候选人打分排名，帮你挑人 |
| **dry-run / 预览** | 只演示不真做，确认无误用的 |
| **简历附件** | 候选人发来的简历文件（PDF），打分主要看它 |
